# Glitch & Chromatic Aberration Tool

A browser-based image effect tool for applying **glitch**, **chromatic aberration**, **RGB split**, **VHS distortion**, **scanlines**, and **digital noise** to images.

## Features

- Apply chromatic aberration and RGB channel offset
- Add glitch distortion and VHS-style interference
- Tune effect intensity and offset values
- Preview changes in real time
- Export processed images
- Runs directly in the browser

## Keywords

glitch effect tool, chromatic aberration tool, RGB split effect, VHS effect generator, photo glitch editor, browser image effects, online image distortion tool

## What This Tool Does

This tool lets you upload an image and create visual distortion effects such as:

- chromatic aberration
- RGB split
- glitch noise
- scanlines
- digital offset artifacts
- retro VHS-style image degradation

It is useful for designers, artists, content creators, and developers who want fast browser-based image styling.

## Use Cases

- Create poster and album-cover effects
- Add retro digital distortion to photos
- Generate social media visuals
- Prototype image-processing effects for design tools
- Build creative coding and web graphics workflows

## How It Works

1. Upload an image
2. Adjust glitch and chromatic aberration settings
3. Preview the output
4. Export the final image
